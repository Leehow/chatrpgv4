# coc7 skill pack

The L2 rule-craft skill pack for `coc7` (see `docs/ruleset-contract.md` §7):
`coc-rules-engine`, `coc-sanity`, `coc-combat`, `coc-chase`, `coc-magic`,
`coc-character`, `coc-mythos-reference`, `coc-development`. Generic protocol
skills (mode activation, play loop, director, campaign state, export,
bootstrap, scenario import, pdf ingest) stay kernel-level at
`plugins/coc-keeper/skills/` and load this pack by reference.
